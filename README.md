# LeosModule

## What is LeosModule?

This is a test module, which uses [Jupyter Notebook](https://jupyter.org/try-jupyter/lab/).
It uses a curated database of **HUMAN** interaction pairs and takes into account that ligands or receptors may consist of multiple subunits.
